<objective>
Add per-physical-machine hostname tracking + apply via `$XDG_STATE_HOME/dotfiles/hostname`, and a `task hostname:set -- <name>` rename command. Hostname is per-physical-machine state (JMBP vs JAIR both consume the same `personal-laptop.toml`), so it lives alongside `machine` and `resolved.json` in `$XDG_STATE_HOME/dotfiles/`, NOT in the manifest. File existence is the gate; no new feature flag.

Purpose: stop hand-running `sethostname` after every fresh-install; make hostname re-application idempotent and observable in the same way as macOS defaults.
Output: `os/hostname.zsh`, `taskfiles/hostname.yml`, refactored `shell/functions/sethostname.zsh`, wired install + validate pipelines.

3. `apply_hostname <name>` — validates via `validate_hostname_name`; runs the four sudo commands from the existing `sethostname` function in the same order (ComputerName, HostName, LocalHostName, SMB NetBIOSName) but uses `>&2` redirection for sudo password prompts if needed (let sudo print to the controlling tty as normal). 4. `verify_hostname <name>` — compares `scutil --get LocalHostName` to `$1`; returns 0 on match, 1 on mismatch; silent (no output) so `status:` blocks can pipe to `/dev/null` cleanly.

Also expose a state-file path helper:

5. `hostname_state_file` — echoes `"${XDG_STATE_HOME:-$HOME/.local/state}/dotfiles/hostname"`. No side effects.

6. `write_hostname_state_file <name>` — validates the name, then mkdir -p the parent dir, then `mktemp + trap rm + printf '%s\n' "$name" > $tmp + mv` (mirror `resolver.zsh::resolve_manifest` exactly: `trap 'rm -f "$tmp"' EXIT INT TERM` BEFORE the pipeline, `trap - EXIT INT TERM` after the successful `mv`). NetBIOS is fire-and-forget.

NO interactive prompts. NO trapping `SIGINT` mid-sudo (let sudo handle its own tty cleanup).

   ```
   function sethostname() {
     local name="${1:-}"
     # Locate the library: DOTFILEDIR if exported, else derive from this
     # file's own path (zsh ${(%):-%N} gives the sourced file path), else
     # fall back to $XDG_CONFIG_HOME/dotfiles.
     local lib=""
     if [[ -n "${DOTFILEDIR:-}" && -f "${DOTFILEDIR}/os/hostname.zsh" ]]; then
       lib="${DOTFILEDIR}/os/hostname.zsh"
     else
       local self_dir="${${(%):-%N}:A:h}"
       # this file: shell/functions/sethostname.zsh -> repo root is ../..
       local repo_root="${self_dir:h:h}"
       if [[ -f "${repo_root}/os/hostname.zsh" ]]; then
         lib="${repo_root}/os/hostname.zsh"
         export DOTFILEDIR="${repo_root}"
       elif [[ -f "${XDG_CONFIG_HOME:-$HOME/.config}/dotfiles/os/hostname.zsh" ]]; then
         lib="${XDG_CONFIG_HOME:-$HOME/.config}/dotfiles/os/hostname.zsh"
         export DOTFILEDIR="${XDG_CONFIG_HOME:-$HOME/.config}/dotfiles"
       fi
     fi
     if [[ -z "$lib" ]]; then
       echo "sethostname: cannot locate os/hostname.zsh (export DOTFILEDIR)" >&2

     source "$lib"
     write_hostname_state_file "$name" || return $?
     apply_hostname "$name" || return $?
   }
   ```
