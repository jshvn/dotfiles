<objective>
Migrate the manifest TOML layer from the legacy flat `extra_packages = [...]` shape to the
typed sub-table shape per D-03: `[packages.brew.extra_packages]` with three sub-arrays
(`formulae`, `casks`, `mas`). Migrate `defaults.toml` and the four machine manifests
(personal-laptop, work-laptop, server-1, server-2). Populate personal-laptop and work-laptop
with their full v1 cask + MAS inventory (sourced from `install/Brewfile-personal.rb` and
`install/Brewfile-work.rb`) as typed objects with mandatory `verify` fields.

Purpose: Implements PKGS-04 (manifest typed-bucket extras schema) and PKGS-05 (server
machines decline GUI by keeping `bundles = ["core"]`). Upstream of the composer (Plan 03)
which reads `.packages.brew.extra_packages.{formulae,casks,mas}` from `resolved.json` --
the composer cannot ship until this schema migration lands.

Output: Five updated TOML files. No bundle file changes (Plan 01 owns those). No resolver
changes (Plan 03 extends the merge pipeline). No taskfile changes (Plan 04).
</objective>
