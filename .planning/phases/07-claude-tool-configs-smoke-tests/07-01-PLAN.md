Pre-Phase-7 repo cleanup and canonical-doc wording amend.

Purpose: Phase 7 D-02 establishes that the dotfiles repo owns only hand-authored Claude assets and the seven repo-owned zsh hooks. Output: A clean `claude/` tree containing only repo-owned hand-authored assets, `.gitignore` patterns that exclude future GSD-installer outputs, and REQUIREMENTS.md + ROADMAP.md wording aligned with D-09.
</objective>
