Port the seven v1 tool config sources into per-tool subdirectories under `configs/`, with destination-matching basenames per D-06, and write the per-tool READMEs plus the aggregate `configs/README.md` body. This plan does NOT touch `taskfiles/links.yml` — Plan 06 owns the symlink registration. This plan ONLY lands the content + READMEs; Plan 06 wires the symlinks.
