## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user shell → `task links:reconcile -- --remove` | interactive y/N input from TTY; must be TTY-gated to prevent script injection |
| user shell → `task cutover:ack -- <name>` | CLI_ARGS user input becomes part of a state-file write; must be regex-validated and indirected via env var |
| `links:reconcile -- --remove` → filesystem | filesystem mutation (symlink deletion); MUST use `unlink` (only removes named symlink) not `rm` |
| EXPECTED_TARGETS parent dir set → find walk | bounded surface (XDG dirs + ZDOTDIR); no recursive scan of $HOME |
| install: pipeline → links:reconcile warn-only | pipeline must not abort on orphan presence; --warn-only mode swallows non-zero exit |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-07 | Tampering | `links:reconcile -- --remove` destructive operation | mitigate | TTY-gate (`[[ -t 0 ]]` per 08-RESEARCH.md security domain); y/N prompt with default N; only `unlink` used (never `rm`, never `rm -rf`); operates only on symlinks identified as orphans via the EXPECTED_TARGETS prefix check |
| T-08-08 | Tampering | `cutover:ack` CLI_ARGS injection | mitigate | env: CLI_ARGS_ENV indirection (Pattern from manifest:setup line 118-119); regex validation `^[a-z0-9_][a-z0-9_-]*$` in preconditions; active-machine match enforcement |
| T-08-09 | Elevation of privilege | `cutover:ack` writes to $XDG_STATE_HOME | accept | $XDG_STATE_HOME is user-owned; sentinel file has no setuid/sticky; reader uses identical path; no root-owned files involved |
| T-08-10 | Spoofing | install-time orphan warn could be suppressed | accept | --warn-only mode prints to stderr via warn() which is always visible; user can disable by setting DOTFILES_MESSAGES_QUIET — out of scope for v1 |
| T-08-11 | Repudiation | reconcile --remove silently deletes | mitigate | y/N prompt for each orphan; magic words only (`y`/`yes`); anything else (including empty/Enter) skips; success message printed for each removal |
| T-08-SC | Tampering | npm/pip/cargo installs | n/a | This plan installs no packages |
